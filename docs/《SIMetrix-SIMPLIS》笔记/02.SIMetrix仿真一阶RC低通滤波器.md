## 截止频率计算

&#8194;&#8194;&#8194;&#8194;假设电阻为1K，电容为1uF，截止频率计算如下：


$$ f=\frac{1}{2πRC} =\frac{1}{2π\bullet 1000 \bullet 10^{-6} } =159.1549Hz $$ 



## 仿真步骤

1. 新建SIMetrix工程文件；

<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像1.png"/></div>

2. 放置元器件（放置元器件时，F5快捷键可以旋转元器件）；

+ 放置电压源、电阻和电容；

<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像2.png"/></div>

+ 在RC之间放置一个电压探头，按快捷键B，也可以从菜单[Place] — [Probe] — [Voltage Probe]中放置；

<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像6.png"/></div>

在R上放置一个[Bode Plot Probe]（放置时，取消全部的默认测试功能）；

<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像7.png"/></div>

3. 保存工程文件；

<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像3.png"/></div>

4. 修改器件参数；

+ V1是5V电源，把Enable AC也选中，表示V1同时也是个频率源，允许进行AC分析；

+ R1是1K电阻，C1是电容，把电容初始电压设置为0；

<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像5.png"/></div>

5. 仿真设置；

点[Simulator] — [Choose Analysis] 进行分析设置

+ 勾选中Transient和AC, 在Transient栏中 Stop time设置10ms,表示仿真时间为10ms；
+ [Advanced Options]中可以设置步长，步长时间越小越精确，但是速度也越慢，这里设置步长为20us；
<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像9.png"/></div>
+ 然后在AC栏设置一下AC分析的起始扫描频率为1Hz和终止频率为1MHz；
<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像10.png"/></div>
+ 点击[Simulator]菜单下的[Run Schematic]，或者按F9运行分析。
<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像11.png"/></div>
6. 仿真结果


+ AC分析和瞬态分析(Transient)的结果如下：截止频率为**160Hz**。

<div align="center"><img src="/blog/blog_image/SIMetrix仿真一阶RC低通滤波器/图像8.png"/></div>
