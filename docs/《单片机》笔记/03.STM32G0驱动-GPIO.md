本次实验目的是初始化STM32G071RB的时钟和每500ms翻转一次PA5的LED。

## 关键程序


```c
  /* Infinite loop */
  /* USER CODE BEGIN WHILE */
  while (1)
  {
    /* USER CODE END WHILE */

    /* USER CODE BEGIN 3 */
    HAL_GPIO_TogglePin(GPIOA, GPIO_PIN_5);
    HAL_Delay(500);
  }
  /* USER CODE END 3 */

```

## 测试结果

实验结果：PA5的LED会每500ms翻转一次。
