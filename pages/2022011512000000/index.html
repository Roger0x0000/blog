模拟滤波器

利用SIMetrix仿真一阶RC低通滤波器，生成bode图。

输出电压与输入电压之比

Au=UoUi=1jωCR+1jωC=11+jωRC (1) Au=\frac{Uo}{Ui}=\frac{\frac{1}{jωC}}{R+\frac{1}{jωC}}=\frac{1}{1+jωRC} \text { (1)} Au=UiUo​=R+jωC1​jωC1​​=1+jωRC1​ (1)

时间常数τ(Tau):

τ=RC τ=RC τ=RC

令ω0(Omega):

ω0=1τ ω_0 = \frac{1}{τ} ω0​=τ1​

特征频率

f0=ω02π=12πτ=12πRC f_0=\frac{ω_0}{2π}= \frac{1}{2πτ}=\frac{1}{2πRC} f0​=2πω0​​=2πτ1​=2πRC1​

代入上式(1)可得

Au=11+jωRC=11+jωω0=11+jff0 Au=\frac{1}{1+jωRC}=\frac{1}{1+j\frac{ω}{ω_0}}=\frac{1}{1+j\frac{f}{f_0}} Au=1+jωRC1​=1+jω0​ω​1​=1+jf0​f​1​

幅频特性

∣Au∣=11+(ff0)2 |Au|=\frac{1}{\sqrt{1+ \left( \frac{f}{f_0}\right)^2 }} ∣Au∣=1+(f0​f​)2​1​

相频特性φ(Phi)

φ=−arctanff0 φ=-arctan\frac{f}{f_0} φ=−arctanf0​f​

Excel中计算

f=0,20lg∣Au∣=0dB f=0, 20lg|Au|=0dB f=0,20lg∣Au∣=0dB
f=f0,20lg∣Au∣≈−3dB f=f_0, 20lg|Au|≈-3dB f=f0​,20lg∣Au∣≈−3dB
f=10f0,20lg∣Au∣≈−20dB f=10f_0, 20lg|Au|≈-20dB f=10f0​,20lg∣Au∣≈−20dB
f=100f0,20lg∣Au∣≈−40dB f=100f_0, 20lg|Au|≈-40dB f=100f0​,20lg∣Au∣≈−40dB
注释 $$ f=0, φ=-\frac{ATAN(\frac{f}{f_0})*180}{PI()} =0° $$ $$ f=f_0, φ=-\frac{ATAN(\frac{f}{f_0})*180}{PI()} =-45° $$ $$ f=10f_0, φ=-\frac{ATAN(\frac{f}{f_0})*180}{PI()} =-84.2894° $$ $$ f=100f_0, φ=-\frac{ATAN(\frac{f}{f_0})*180}{PI()} =-89.4271° $$

传递函数

推导其传递函数可得如下公式：

Hp=1sCR+1sC=11+sRC (2) H_p=\frac{\frac{1}{sC}}{R+\frac{1}{sC}}=\frac{1}{1+sRC} \text { (2)} Hp​=R+sC1​sC1​​=1+sRC1​ (2)

获得上面Bode图的Python代码如下：


import numpy as np
import matplotlib.pyplot as plt
import control as ctrl

s = ctrl.tf('s')

C = 1e-6    #uF
R = 1e3     #1k
fc = 1/(2*np.pi*R*C)

Hp = 1/(1+(R*C)*s)

f = np.logspace(-1, 6, 1000)
w = 2 * np.pi * f

mag,phase,omega=ctrl.bode_plot(Hp,w,dB=True,Hz=True,deg=True,Plot=True,label='Hp')

plt.legend()
plt.show()

pole=ctrl.pole(Hp)

print("fc =", fc, "Hz")                 # fc = 159.15494309189535 Hz
print("pole =", pole/(2*np.pi), "Hz")   # pole = [-159.15494309] Hz

数字滤波器

对公式（2）用双线性变换进行离散化处理，将s=2Ts1−z−11+z−1s=\frac{2}{T_s}{\frac{1-z^{-1}}{1+z^{-1}}}s=Ts​2​1+z−11−z−1​带入得到zzz域传递函数 - 公式（3）。

Hpd=1+z−1Ts+2τTs−2τ−TsTsz−1 (3) H_{pd}=\frac{1+z^{-1}}{\frac{T_s+2\tau}{T_s}-\frac{2\tau-T_s}{T_s}z^{-1}} \text { (3)} Hpd​=Ts​Ts​+2τ​−Ts​2τ−Ts​​z−11+z−1​ (3)

假设fc=ωc2π=12πτ=12πRC=159.15Hzf_c=\frac{ω_c}{2π}= \frac{1}{2πτ}=\frac{1}{2πRC}=159.15Hzfc​=2πωc​​=2πτ1​=2πRC1​=159.15Hz ，fs=100kHzf_s=100kHzfs​=100kHz，Ts=1fsT_s=\frac{1}{f_s}Ts​=fs​1​，可以画出连续域与离散域的Bode图对比如下，因采样频率为100kHz，所以根据滴滴香浓定理转换为离散域后只有低于50kHz能够进行复现。也容易观察到经双线性变换离散化处理后，转化为离散域的截止频率仍为159.15Hz，相位曲线和连续域基本一致，增益曲线仅在接近fs2\frac{f_s}{2}2fs​​附近幅值衰减更大。

获得上面Bode图的Python代码如下：


import numpy as np
import matplotlib.pyplot as plt
import control as ctrl

############ continuous s transfer function. ############
s = ctrl.tf('s')

C = 1e-6    #uF
R = 1e3     #1k
fc = 1/(2*np.pi*R*C) # low pass filter cut-off frequency 

Hp = 1/(1+(R*C)*s)


f = np.logspace(-1, 6, 1000)
w = 2 * np.pi * f

mag,phase,omega=ctrl.bode_plot(Hp,w,dB=True,Hz=True,deg=True,Plot=True, grid=True, label='Hp')

pole=ctrl.pole(Hp)

print("pole =", pole/(2*np.pi), "Hz")

############ discrete z transfer function. ############
z = ctrl.tf('z')

fs = 100e3    # sample frequency is 100kHz, fc is still 1kHz
Ts = 1/fs

t = 1/(2*np.pi*fc)

Hpd = (1+z**(-1))/((Ts+2*t)/Ts - (2*t-Ts)/Ts*z**(-1))
print(Hpd)
Hpd.dt = Ts

mag,phase,omega=ctrl.bode_plot(Hpd,w,dB=True,Hz=True,deg=True,Plot=True, grid=True, label='Hpd')

plt.legend()
plt.show()

差分方程

公式（3）进一步转化为软件可实现的差分方程公式（4）。

Hpd=1+z−1Ts+2τTs−2τ−TsTsz−1 (公式3) H_{pd}=\frac{1+z^{-1}}{\frac{T_s+2\tau}{T_s}-\frac{2\tau-T_s}{T_s}z^{-1}} \text { (公式3)} Hpd​=Ts​Ts​+2τ​−Ts​2τ−Ts​​z−11+z−1​ (公式3)
y[n]=TsTs+2τ∙x[n]+TsTs+2τ∙x[n−1]+2τ−TsTs+2τ∙y[n−1] y[n]=\frac{T_s}{T_s+2\tau} \bullet x[n] + \frac{T_s}{T_s+2\tau} \bullet x[n-1] + \frac{2\tau-Ts}{T_s+2\tau} \bullet y[n-1] y[n]=Ts​+2τTs​​∙x[n]+Ts​+2τTs​​∙x[n−1]+Ts​+2τ2τ−Ts​∙y[n−1]
=kxn∙x[n]+kxn∙x[n−1]+kyn∙y[n] (公式4)=k_{xn}\bullet x[n] + k_{xn}\bullet x[n-1] + k_{yn}\bullet y[n] \text { (公式4)}=kxn​∙x[n]+kxn​∙x[n−1]+kyn​∙y[n] (公式4)

其中 kxn=TsTs+2τk_{xn}=\frac{T_s}{T_s+2\tau}kxn​=Ts​+2τTs​​,kyn=2τ−TsTs+2τk_{yn}=\frac{2\tau-T_s}{T_s+2\tau}kyn​=Ts​+2τ2τ−Ts​​

差分方程系数的获取，根据前述假设条件fc=12πτ=159.15Hzf_{c}=\frac{1}{2π\tau}=159.15Hzfc​=2πτ1​=159.15Hz,fs=100kHzf_{s}=100kHzfs​=100kHz,Ts=1fsT_{s}=\frac{1}{f_s}Ts​=fs​1​可以得到该实例滤波器系数如下：

kxn=TsTs+2τ=0.004975124378109453 k_{xn}=\frac{T_s}{T_s+2\tau} = 0.004975124378109453kxn​=Ts​+2τTs​​=0.004975124378109453
kyn=2τ−TsTs+2τ=0.990049751243781 k_{yn}=\frac{2\tau-T_s}{T_s+2\tau} = 0.990049751243781kyn​=Ts​+2τ2τ−Ts​​=0.990049751243781

求得上面系数的Python代码如下：


import numpy as np

C = 1e-6    #uF
R = 1e3     #1k
fc = 1/(2*np.pi*R*C) # low pass filter cut-off frequency 

fs = 100e3    # sample frequency is 100kHz
Ts = 1/fs

t = 1/(2*np.pi*fc)

Kxn = Ts/(Ts+2*t)
Kyn = (2*t-Ts)/(Ts+2*t)

print("fc =", fc)  
print("Kxn =", Kxn, "Kyn=", Kyn)

#fc = 159.15494309189535
#Kxn = 0.004975124378109453 Kyn= 0.990049751243781

y[n]=A∙x[n]+(1−A)∙y[n−1] y[n]= A \bullet x[n] + (1-A) \bullet y[n-1] y[n]=A∙x[n]+(1−A)∙y[n−1]
A=11+12∙π∙Ts∙fc A =\frac{1}{1+\frac{1}{2\bullet π\bullet T_s \bullet f_c}}A=1+2∙π∙Ts​∙fc​1​1​

A为滤波系数，滤波系数的大小与采样周期TsT_sTs​和滤波器的截止频率fcf_cfc​有关。

有公式可知一阶滤波器与当前的输入值和上一次的输出值有关；滤波系数越小，滤波结果越平稳，但是灵敏度越低；滤波系数越大，灵敏度越高，但是滤波结果越不稳定。

  • 一阶向后差分法


import numpy as np
import matplotlib.pyplot as plt
 
Ts=0.003                     #采样时间
Fl=5                         #截止频率
a=1/(1+1/(2*np.pi*Ts*Fl))    #滤波系数
 
print(a)                         #打印滤波计算出的值
x=np.linspace(-np.pi,np.pi,2000) #在[-pi,pi]区间上分割正2000个点
                                 # 可以理解为信号采样时间为 2*pi/2000s
data1=np.zeros_like(x)           #输入信号
data=np.zeros_like(x)            #输入信号
y1=np.zeros_like(x)              #一阶滤波输出
for i in range(len(x)):
    #幅值为1 频率为 1Hz的信号  + 幅值为0.5 频率为100Hz的干扰信号+ 幅值为0.2 频率为6Hz的干扰信号
    data =np.sin( 2 * np.pi * x)+0.2*np.sin(6* 2 * np.pi * x)\
             +0.5*np.sin(100* 2 * np.pi * x)
    data1[i]=np.sin( 2 * np.pi * x[i])
    if(i>0):
        y1[i] = a * data[i] + (1 - a) * y1[i - 1]
    else:
        y1[0] = data[0]
 
#绘制被干扰的信号  + 一阶滤波后的信号
 
plt.subplot(2, 1, 1)
plt.plot(x,data ,label='sig+noise')
plt.plot(x,y1,   'r',label='first order')
plt.title("Lowpass Filter Frequency Response")
plt.xlabel('Time [sec]')
plt.grid()
plt.legend()
#原信号  + 一阶滤波后的信号
plt.subplot(2, 1, 2)
plt.plot(x,data1 ,label='sig')
plt.plot(x,y1,   'r',label='first order')
plt.xlabel('Time [sec]')
plt.grid()
plt.legend()
 
plt.show()
