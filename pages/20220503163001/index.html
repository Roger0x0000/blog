STM32G0驱动-PWM

本次实验目的是初始化STM32G071RB的定时器和输出两路25KHz的PWM。

原理图

定时器配置
Period=64MHz/25KHz=2560Period=64MHz/25KHz =2560Period=64MHz/25KHz=2560

关键程序

在main.c初始化中添加HAL_TIM_PWM_Start()和__HAL_TIM_SetCompare()函数。

  /* USER CODE BEGIN 2 */
  /* Start channel 1  25KHz (64MHz/2560= 25KHz)*/
  if (HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_1) != HAL_OK)
  {
    /* PWM Generation Error */
    Error_Handler();
  }
  /* Start channel 2  25KHz (64MHz/2560= 25KHz)*/
  if (HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_2) != HAL_OK)
  {
    /* PWM Generation Error */
    Error_Handler();
  }
  
  __HAL_TIM_SetCompare(&htim2, TIM_CHANNEL_1, 2560>>1); //50%Duty
  __HAL_TIM_SetCompare(&htim2, TIM_CHANNEL_2, 2560>>2); //25%Duty
  /* USER CODE END 2 */

  /* Infinite loop */
  /* USER CODE BEGIN WHILE */
  while (1)
  {
    /* USER CODE END WHILE */

    /* USER CODE BEGIN 3 */
		
  }
  /* USER CODE END 3 */
