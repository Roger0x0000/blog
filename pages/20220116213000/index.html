传递函数推导

    借助Python可以方便推导出补偿器的传递函数：

import sympy as sp

#两电阻并联总值
def parallel(a,b):
    return (a*b)/(a+b)


R1,R2,R3,C1,C2,C3,s = sp.symbols('R1 R2 R3 C1 C2 C3 s')
 
Zf = parallel(R2 + 1/(C1*s), 1/(C2*s))
Zi = parallel(R3 + 1/(C3*s), R2)

H3 = sp.factor(Zf/Zi)
print("H3 = ",H3)  

#H3 =  (C1*R2*s + 1)*(C3*R2*s + C3*R3*s + 1)/(R2*s*(C3*R3*s + 1)*(C1*C2*R2*s + C1 + C2))

案例

    假设需要设计一个补偿器， 它在5kHz位置的增益为-10dB， 且在此处的相位提升量为145度，这样的补偿器该如何设置呢？

  • 计算在穿越频率处提供幅值增益-10dB的十进制增益：
G=10(−1020)=0.316G = 10^{(-\frac{10}{20})}=0.316G=10(−2010​)=0.316
  • 求取K因子
K=[Tan((Boost4)+45°)]2=[Tan((145°4)+45°)]2=[Tan(81.25∙π180)]2=42.21 K = {[Tan((\frac{Boost}{4}) + 45°)]}^2 = [Tan((\frac{145°}{4}) + 45°)]^2=[Tan(\frac{81.25\bullet π}{180})]^2 = 42.21 K=[Tan((4Boost​)+45°)]2=[Tan((4145°​)+45°)]2=[Tan(18081.25∙π​)]2=42.21
  • 求取fp1f_{p1}fp1​和fp2f_{p2}fp2​
fp1=fp2=K∙fc=6.49∗5k=32.48kHzf_{p1}=f_{p2}=\sqrt{K}\bullet f_{c} = 6.49*5k=32.48kHzfp1​=fp2​=K​∙fc​=6.49∗5k=32.48kHz

  • 求取fz1f_{z1}fz1​和fz2f_{z2}fz2​

    fz1=fz2=fcK=5k/6.49=770Hzf_{z1}=f_{z2}=\frac{f_{c}}{\sqrt{K}} = 5k/6.49=770Hzfz1​=fz2​=K​fc​​=5k/6.49=770Hz

  • 假设R1=10kohm，按照K因子法计算元器件参数：

C2=12∗π∗fc∗G∗R1=12∗π∗5000∗0.316∗10000=10nFC2 =\frac{1}{2*π*f_c*G*R1} = \frac{1}{2*π*5000*0.316*10000} = 10nFC2=2∗π∗fc​∗G∗R11​=2∗π∗5000∗0.316∗100001​=10nF
C1=C2∗(K−1)=10nF∗(42.21−1)=415nFC1 =C2*(K-1) =10nF*(42.21-1) = 415nF C1=C2∗(K−1)=10nF∗(42.21−1)=415nF
R2=K2∗π∗fc∗C1=42.212∗π∗5000∗415nF=498ΩR2 =\frac{\sqrt{K}}{2*π*f_c*C1} = \frac{\sqrt{42.21}}{2*π*5000*415nF} = 498ΩR2=2∗π∗fc​∗C1K​​=2∗π∗5000∗415nF42.21​​=498Ω
C1=R1K−1=1000042.21−1=242ΩC1 =\frac{R1}{K-1} =\frac{10000}{42.21-1} = 242Ω C1=K−1R1​=42.21−110000​=242Ω
C3=12∗π∗fc∗K∗R3=20nFC3 =\frac{1}{2*π*f_c*\sqrt{K}*R3} =20nFC3=2∗π∗fc​∗K​∗R31​=20nF

借助Python可以方便计算出补偿器的参数：


import numpy as np
import matplotlib.pyplot as plt
import control as ctrl
import math


#推荐使用 control 0.8.3

s = ctrl.tf('s')


C1 = 415*math.pow(10.0,-9)  #415nF
C2 = 10*math.pow(10.0,-9)   #10nF
C3 = 20*math.pow(10.0,-9)   #20nF
R1 = 10e3 #10K
R2 = 498    #498 ohm
R3 = 242    #242 ohm

f_p0 = 1/(2*np.pi*R1*(C1+C2))
f_z1 = 1/(2*np.pi*R2*C1)
f_z2 = 1/(2*np.pi*(R3+R1)*C3)
f_p1 = 1/(2*np.pi*R3*C3)
f_p2 = 1/(2*np.pi*R2*((C1*C2)/(C1+C2))) 

print("f_p0 =", f_p0, "Hz") 
print("f_z1 =", f_z1, "Hz") 
print("f_z2 =", f_z2, "Hz") 
print("f_p1 =", f_p1, "Hz") 
print("f_p2 =", f_p2, "Hz") 

# f_p0 = 37.448221903975366 Hz
# f_z1 = 770.0921425068725 Hz
# f_z2 = 776.9719932234688 Hz
# f_p1 = 32883.25270493705 Hz
# f_p2 = 32728.91605654208 Hz


w_p0 = 2*np.pi*f_p0
w_z1 = 2*np.pi*f_z1
w_z2 = 2*np.pi*f_z2
w_p1 = 2*np.pi*f_p1
w_p2 = 2*np.pi*f_p2


H3 = -(w_p0/s)*(1+s/w_z1)*(1+s/w_z2)/((1+s/w_p1)*(1+s/w_p2))

f = np.logspace(-1, 6, 1000)
w = 2 * np.pi * f

mag,phase,omega=ctrl.bode_plot(H3,w,dB=True,Hz=True,deg=True,Plot=True,label='H3')

plt.legend()
plt.show()
