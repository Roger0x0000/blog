                                                                              # 基于连续系统设计的LLC谐振变换器

                                                                              本节主要介绍如何使用PLECS搭建一个闭环LLC谐振变换。

                                                                              # LLC谐振变换基本工作模态

                                                                              LLC谐振变换具体的工作模态，可以观看上科大的傅旻帆老师的视频电力电子技术(软开关、串联谐振、并联谐振、LLC变换器、准谐振、PWM谐振技术、高频逆变整流、无线电能传输)open in new window

                                                                              • ZCS区域(ZCS Region)：增益曲线峰值点的左侧，为容性区域。

                                                                              • ZVS区域(ZVS Region)：增益曲线峰值点的右侧，为感性区域。

                                                                              • 区域1(Region 1):为感性区域。

                                                                              • 区域2(Region 2):为感性区域。

                                                                              • 区域3(Region 3):为容性区域。

                                                                              • a：工作频率等于串联谐振频率fs=f0f_s = f_0fs​=f0​，Lm不参与谐振，增益等于1；所有Q值曲线都经过这一点，不管负载如何变化，输出电压都不会变化，调整率好。

                                                                              • b：工作频率小于串联谐振频率fp<fsw<f0f_p < f_{sw}<f_0fp​<fsw​<f0​，Lm参与谐振，增益大于1；增益曲线比较陡，增益需要做大的变化，频率要做很小的变化，调整率好。

                                                                              • c：工作频率大于串联谐振频率fs>f0f_s > f_0fs​>f0​，Lm不参与谐振，增益小于1；增益曲线平缓，增益需要做小的变化，频率要做很大的变化，调整率不好。

                                                                              直流增益：

                                                                              M=n⋅VoVge2=2⋅n⋅VoVge M = \frac{n\cdot V_o}{\frac{V_{ge}}{2}} = \frac{2 \cdot n \cdot V_o}{V_{ge}} M=2Vge​​n⋅Vo​​=Vge​2⋅n⋅Vo​​

                                                                              串联谐振频率：

                                                                              f0=12⋅πLr⋅Cr f_0 = \frac{1}{2\cdot \pi \sqrt{L_r \cdot C_r}} f0​=2⋅πLr​⋅Cr​​1​

                                                                              并联谐振频率：

                                                                              fp=12⋅π(Lr+Lm)⋅Cr f_p = \frac{1}{2\cdot \pi \sqrt{(L_r + L_m) \cdot C_r}} fp​=2⋅π(Lr​+Lm​)⋅Cr​​1​

                                                                              品质因数：

                                                                              Q=LrCrRe Q = \frac{\sqrt{\frac{L_r}{C_r}}}{R_e} Q=Re​Cr​Lr​​​​

                                                                              归一化频率：

                                                                              fn=ff0 f_n = \frac{f}{f_0} fn​=f0​f​

                                                                              电感比：

                                                                              Ln=LmLr L_n = \frac{L_m}{L_r} Ln​=Lr​Lm​​

                                                                              原边交流等效电阻ReR_eRe​（上图的Ro′R_o'Ro′​）：

                                                                              Re=8⋅n2π2⋅VoIo R_e = \frac{8\cdot n^2}{\pi ^2} \cdot \frac{V_o}{I_o} Re​=π28⋅n2​⋅Io​Vo​​

                                                                              直流增益曲线：

                                                                              M=Ln⋅fn2[(1+Ln)⋅fn2−1]2+[Q⋅Ln⋅fn⋅(fn2−1)]2 M = \frac{L_n \cdot f_n^2}{\sqrt{[(1+L_n)\cdot f_n^2- 1]^2 +[Q\cdot L_n \cdot f_n \cdot(f_n^2-1)]^2}} M=[(1+Ln​)⋅fn2​−1]2+[Q⋅Ln​⋅fn​⋅(fn2​−1)]2​Ln​⋅fn2​​

                                                                              输出电压：

                                                                              Vo=Mg(fn,Ln,Qe)⋅1n⋅Vge2 V_{o} = M_{g}(f_n,L_n,Q_e)\cdot \frac{1}{n}\cdot \frac{V_{ge}}{2} Vo​=Mg​(fn​,Ln​,Qe​)⋅n1​⋅2Vge​​

                                                                              假设Ln=5.0L_n = 5.0Ln​=5.0，我们可以画出以下增益曲线。

                                                                              获得上面增益曲线的Pythond代码如下。

                                                                              详情
                                                                              
import numpy as np
import matplotlib.pyplot as plt
import control as ctrl
import scipy

print("Scipy version is ",scipy.__version__) 
print("Control version is ",ctrl.__version__)

def LLC_Gain_Chart(fLn = 5.0):

    Fn = np.arange(0.0, 3.0, 0.0001)#(start, stop, step, dtype = None)
    Ln = fLn
    
    for i in range(1, 10, 2):
        Q = float(i * 0.1)
        #M = 1/(np.sqrt( (1+1/Ln - 1/(Ln*Fn * Fn))*(1+1/Ln - 1/(Ln*Fn * Fn))+  Q *Q *(Fn-1/Fn)*(Fn-1/Fn)  ))
        M = (Ln * Fn * Fn) / (np.sqrt(((1 + Ln) * Fn * Fn - 1) * ((1 + Ln) * Fn * Fn - 1) + (Q * Ln * Fn * (Fn * Fn - 1)) * (Q * Ln * Fn * (Fn * Fn - 1))))


        plt.plot(Fn, M, label='Q = ' + '{:.1f}'.format(Q))
        
    plt.title("Ln = " + '{:.1f}'.format(Ln))
    plt.xlim(0, 3)  # 可以自己设置x轴的坐标的范围哦;
    plt.ylim(0, 2)
    plt.ylabel('Gain,'+r'$M{_g}$')  # 设置y轴的label
    plt.xlabel('Normalized Frequency,'+r'$f{_n}$')  # 设置x轴的label
    plt.legend()  # 显示右上角的那个label,
    plt.grid(True)  # 显示网格;
    plt.show()

LLC_Gain_Chart(5.0)

Scipy version is  1.7.3
Control version is  0.8.3
"""

                                                                              # LLC变换器的补偿器计算实例

                                                                              已知变换器电路参数： Vdc = 380V，n_primary = 33，n_secondary = 2，Lr = 55e-6，Lm = 275e-6，Cr = 24e-9，Co = 1300e-6，Vout = 12，Iout = 5，Ro = 12/5，ESR = 15e-3。

                                                                              根据上面的已经参数，可以计算出以下参数。

                                                                              串联谐振频率：

                                                                              f0=12⋅πLr⋅Cr=138.526597kHz f_0 = \frac{1}{2\cdot \pi \sqrt{L_r \cdot C_r}} = 138.526597kHz f0​=2⋅πLr​⋅Cr​​1​=138.526597kHz

                                                                              并联谐振频率：

                                                                              fp=12⋅π(Lr+Lm)⋅Cr=56.553246kHz f_p = \frac{1}{2\cdot \pi \sqrt{(L_r + L_m) \cdot C_r}}=56.553246kHz fp​=2⋅π(Lr​+Lm​)⋅Cr​​1​=56.553246kHz

                                                                              电感比：

                                                                              Ln=LmLr=5.0 L_n = \frac{L_m}{L_r}=5.0 Ln​=Lr​Lm​​=5.0

                                                                              直流增益：

                                                                              M=n⋅VoVge2=2⋅n⋅VoVge=1.0421 M = \frac{n\cdot V_o}{\frac{V_{ge}}{2}} = \frac{2 \cdot n \cdot V_o}{V_{ge}} = 1.0421 M=2Vge​​n⋅Vo​​=Vge​2⋅n⋅Vo​​=1.0421

                                                                              品质因数：

                                                                              Q=LrCrRe=0.0903871 Q = \frac{\sqrt{\frac{L_r}{C_r}}}{R_e} = 0.0903871 Q=Re​Cr​Lr​​​​=0.0903871

                                                                              将LnL_nLn​、QQQ和MMM代入直流增益曲线函数：

                                                                              M=Ln⋅fn2[(1+Ln)⋅fn2−1]2+[Q⋅Ln⋅fn⋅(fn2−1)]2 M = \frac{L_n \cdot f_n^2}{\sqrt{[(1+L_n)\cdot f_n^2- 1]^2 +[Q\cdot L_n \cdot f_n \cdot(f_n^2-1)]^2}} M=[(1+Ln​)⋅fn2​−1]2+[Q⋅Ln​⋅fn​⋅(fn2​−1)]2​Ln​⋅fn2​​

                                                                              求得归一化频率：

                                                                              fn=fswf0=0.9118275956819146 f_n = \frac{f_{sw}}{f_0}=0.9118275956819146 fn​=f0​fsw​​=0.9118275956819146

                                                                              求得满载时的工作频率：

                                                                              fsw=126.312374kHz f_{sw} = 126.312374kHz fsw​=126.312374kHz

                                                                              使用PLECS的仿真结果如下。

                                                                              从伯德图可以看出：电感器和输出电容器的形成的双极点大约为4kHz。输出电容ESR零点大约为8kHz。

                                                                              我们使用Type2补偿器进行补偿。

                                                                              fp0=509.2958Hzf_{p0} = 509.2958Hzfp0​=509.2958Hz用于调节变换器的低频增益和环路带宽，fp1=8200.0Hzf_{p1}=8200.0Hzfp1​=8200.0Hz用于对消变换器的输出电容ESR的零点，fz1=4200.0Hzf_{z1}=4200.0Hzfz1​=4200.0Hz用于对消变换器的双极点的其中一个极点。

                                                                              获得上面Bode图的Python代码如下。

                                                                              详情
                                                                              
import numpy as np
import matplotlib.pyplot as plt
import control as ctrl


#推荐使用 control 0.8.3

s = ctrl.tf('s')

f_p0 = 509.29581789406507
f_z1 = 4200.0
f_p1 = 8200.0

print("f_p0 =", f_p0, "Hz") 
print("f_z1 =", f_z1, "Hz") 
print("f_p1 =", f_p1, "Hz") 


w_p0 = 2*np.pi*f_p0      # DC gain of type 2 compensator
w_z1 = 2*np.pi*f_z1      # frequency of type2 zero
w_p1 = 2*np.pi*f_p1      # frequency of type2 pole1

print("W_p0 =", w_p0, "Hz") 
print("W_z1 =", w_z1, "Hz") 
print("W_p1 =", w_p1, "Hz") 


H2 = (w_p0/s)*(1+s/w_z1)/(1+s/w_p1)

f = np.logspace(1, 6, 1000)
w = 2 * np.pi * f

mag,phase,omega=ctrl.bode_plot(H2,w,dB=True,Hz=True,deg=True,Plot=True,label='Type2')

print(H2)

plt.legend()
plt.show()

"""
f_p0 = 509.2958 Hz
f_z1 = 4200.0 Hz
f_p1 = 8200.0 Hz
W_p0 = 3200.0 Hz
W_z1 = 26389.37829015426 Hz
W_p1 = 51522.119518872605 Hz

  1.649e+08 s + 4.351e+12
--------------------------
2.639e+04 s^2 + 1.36e+09 s

"""

                                                                              上面的Python代码，可以求得Type2的传递函数如下。

                                                                              H(s)=1.649e8⋅s+4.351e122.639e04⋅s2+1.36e09⋅s H(s) = \frac{1.649e8 \cdot s + 4.351e12}{2.639e04 \cdot s^2 + 1.36e09 \cdot s} H(s)=2.639e04⋅s2+1.36e09⋅s1.649e8⋅s+4.351e12​

n_prim = 33;

%Secondary Side

%% Controller

R2 = R1/3;

V_ref = V_out_desired * R2 / (R1 + R2); % reference voltage
